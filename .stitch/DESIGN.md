# Alpha CRM Monochrome Design System

Source of truth for the modern monochrome UI/UX of Project Alpha.

## Visual Language
- **Style**: Minimalist, high contrast, generous whitespace, sharp borders.
- **Atmosphere**: Premium, professional, focused, "Linear-esque".

## Color Palette
| Token | Role | Hex Code |
|:---|:---|:---|
| `primary` | Primary Brand/Text | `#000000` |
| `background` | Page Background | `#FFFFFF` |
| `surface` | Cards/Overlays | `#FAFAFA` |
| `border` | Dividers/Outlines | `#E5E7EB` |
| `muted` | Secondary Text | `#71717A` |
| `accent` | Subtle Focus | `#F4F4F5` |

## Typography
- **Primary**: Inter (variable)
- **Headings**: Tight tracking, semibold/bold.
- **Body**: Regular, legible line-height.
- **Monospace**: JetBrains Mono (for IDs and technical data).

## Components
- **Borders**: 1px solid `#E5E7EB`.
- **Shadows**: Subtle, natural elevation only on modals/drawers.
- **Corners**: Sharp (0px) or slightly rounded (4px) for a modern look.
