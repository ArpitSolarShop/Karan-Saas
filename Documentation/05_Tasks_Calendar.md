# 📋 Tasks, Calendar & Daily Work

Stay organized with **Tasks** and **Calendar** — your personal productivity tools inside the CRM.

---

## Tasks

### What Is a Task?
A task is a to-do item assigned to you (or by you to someone else). Examples:
- "Call Mr. Sharma about the proposal"
- "Send invoice to Acme Corp"
- "Follow up with lead from trade show"

### Viewing Your Tasks
1. Click **Tasks** in the sidebar.
2. You'll see all tasks in a list, sorted by due date.
3. Overdue tasks appear in **red**.
4. Today's tasks appear in **yellow**.
5. Future tasks appear in **green**.

### Creating a Task
1. Click **+ New Task**.
2. Fill in:
   - **Title** — What needs to be done
   - **Due Date** — When it needs to be done
   - **Assigned To** — Who should do it (can be yourself)
   - **Lead** — (Optional) Link it to a specific lead
   - **Notes** — Any additional details
3. Click **Save**.

### Completing a Task
- Click the **checkbox** next to a task to mark it as done.
- Completed tasks move to the "Completed" section.

### Task Tips
- 🔴 **Red tasks** = Overdue! Handle these first.
- Set tasks for **follow-ups** so you never forget a customer.
- Your manager can see your task completion rate.

## Calendar

### Viewing Your Calendar
1. Click **Calendar** in the sidebar.
2. View by **Day**, **Week**, or **Month** using the buttons at the top.

### What Shows on the Calendar?
- **Meetings** — Scheduled meetings with clients.
- **Follow-ups** — Scheduled callback dates.
- **Tasks** — Tasks with due dates.
- **Events** — Company events, webinars, conferences.

### Creating a Calendar Event
1. Click on any date/time slot.
2. Fill in:
   - **Title** — Name of the event
   - **Start & End Time** — When it happens
   - **Description** — Details about the event
   - **Attendees** — People you want to invite
3. Click **Save**.

### Color Coding
| Color | Meaning |
|-------|---------|
| 🔵 Blue | Meetings |
| 🟢 Green | Follow-ups |
| 🟡 Yellow | Tasks |
| 🟣 Purple | Company events |

## Notes

### Adding Notes to a Lead or Deal
1. Open any lead or deal profile.
2. Scroll to the **Notes** section.
3. Type your note.
4. Click **Add Note**.

### Why Notes Matter
- **Memory** — You won't remember every conversation. Notes do.
- **Team collaboration** — Your colleagues can read your notes.
- **Handoffs** — If a lead is reassigned, the new owner has full context.

### Note Best Practices
- Write notes **immediately after** a call or meeting.
- Include **key points**: what was discussed, what was promised, what's next.
- Keep it **concise** but useful.

## Daily Workflow Suggestion

Here's a recommended daily routine:

| Time | Action |
|------|--------|
| 9:00 AM | Check your **Dashboard** for today's overview |
| 9:15 AM | Review **Tasks** — handle overdue ones first |
| 9:30 AM | Check **WhatsApp** for unread messages |
| 10:00 AM | Start **Calling** leads on your list |
| 12:00 PM | Update **Deal stages** after morning calls |
| 2:00 PM | Send **Proposals/Quotes** for qualified deals |
| 4:00 PM | Add **Notes** for all interactions today |
| 5:00 PM | Create **Tasks** for tomorrow's follow-ups |
