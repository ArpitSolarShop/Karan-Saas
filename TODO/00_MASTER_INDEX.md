- **Backend**: NestJS + Prisma
- **Frontend**: Next.js + Tailwind + shadcn/ui
- **Database**: PostgreSQL + Redis
- **Telephony**: Asterisk (ARI/AMI) + Kamailio (SIP proxy)
- **AI**: OpenAI + pgvector + multi-vendor support
- **Infrastructure**: Docker, BullMQ, Meilisearch, MinIO, Prometheus, Grafana
